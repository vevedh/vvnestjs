# vvnestjs-api

## Description

todo-api-for-angular-app

 
 Add tools service  to generate dynamically  a  new  mongoose Table  with Modal Controller Service  and description view with swagger

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
